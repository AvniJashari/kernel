# KERNEL

il progetto cerca di creare un minikernel su architetura RISC V a 64 bit, per la virtualizazione uso QEMU. per far copilare e far partire QEMU basta usare il file run.sh

## dipendenze
rust, con scaricato anche riscv64gc-unknown-none-elf
QEMU, con la virtualizazione per architeture RISC V 64bit
